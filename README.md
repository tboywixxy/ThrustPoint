# ThrustPoint

ThrustPoint is a Next.js app scaffolded with TypeScript, Tailwind CSS, ESLint,
and the App Router.

## Getting Started

Install dependencies if needed:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
